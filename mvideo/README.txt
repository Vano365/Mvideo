В консоли перейдите в папку с проектом.
Установите зависимости - pip install -r requirements.txt
Посмотрите вашу версию Google Chrome и скачать chromdriver под вашу операционную систему с сайта - https://chromedriver.chromium.org/
Запустите парсер комментариев Mvideo в этой же папке проекта - python parser.py
Перейдите в дерикторию с сайтом - cd mvideo, запустите сайт на локальном порту командой python manage.py runserver.
Смотреть, сортировать комментарии можно по ссылке http://localhost:8000/admin (Логин - admin , Пароль - admin)
